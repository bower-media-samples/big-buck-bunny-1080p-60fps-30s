big-buck-bunny-1080p-60fps-30s
==============================

30 first seconds of Big Buck Bunny (1080p@60fps)

This is intended to be used as a sample video for testing (ex: `<video>` tag).

Installation
------------

Install using bower:

    bower install big-buck-bunny-1080p-60fps-30s --save-dev
